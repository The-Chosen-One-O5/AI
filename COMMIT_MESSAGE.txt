Replace ElevenLabs with Edge TTS for free unlimited text-to-speech

This commit replaces the paid Replicate/ElevenLabs TTS implementation with
Microsoft's free Edge TTS service, providing unlimited high-quality voice
synthesis at zero cost.

- Added edge-tts library to requirements.txt
- Replaced generate_audio_from_text() to use Edge TTS
- Set en-US-GuyNeural (deep male) as default voice
- Added 9 voice options (male, female, British)
- Enhanced /audio command with voice selection
- Added /audio list and /audio <voice> commands
- Implemented per-chat voice configuration
- Removed dependency on REPLICATE_API_KEY for TTS
