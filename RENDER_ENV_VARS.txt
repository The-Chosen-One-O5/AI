# Copy these to your Render.com dashboard under Environment tab
# Go to: Dashboard → Your Service → Environment → Add Environment Variable

# ==============================================================================

# ==============================================================================
# NOTES
# ==============================================================================

# 1. PORT is automatically set by Render - DO NOT add it manually
# 2. Never commit these values to git
# 3. Get BOT_TOKEN from @BotFather on Telegram
# 4. Get chat ID by running /chatid command in your group
# 5. See RENDER_DEPLOYMENT.md for API key sources

# ==============================================================================
# HOW TO ADD IN RENDER
# ==============================================================================

# 1. Go to https://dashboard.render.com
# 2. Select your service
# 3. Click "Environment" tab on the left
# 4. Click "Add Environment Variable"
# 5. Enter Key and Value
# 6. Click "Save Changes" (this will trigger a redeploy)
# 7. Repeat for each variable
